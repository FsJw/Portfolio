Walton International

Drupal Front- End Styling

Walton.com was one of the larger projects I was involved in during my internship with SDMG. I was given the task of redesigning the entire website based on designs that their team provided within the given time frame. This experience taught me two things, the first was managing a large project by breaking it down to multiple pieces and estimating how long each piece would take. The second was a lesson on attention to detail given to me by our Director and the rest of the development team. Before this project I viewed design flats as more of a guideline, but I quickly learned that when the designs come from the client themselves it means that the designs should be recreated down to the pixel. Redesigning Waltons website also involved creating new functionality that was not available on the previous version of the site. This meant creating Drupal content types, taxonomy, installing new modules and template editing in order for the new site to function as instructed.

You can view the Walton International website here Walton.com.

Canalta Cares Module

Dynamic Photo Gallery.

The Canalta Cares Drupal Module was my first project in a lead developer position with SDMG. The module itself is a dynamic and interactive photo gallery that uses Facebook’s Graph API to pull photos from the Canalta Hotels Facebook page and the descriptions associated with them. It uses jQuery to create an on hover effect that flips the photo to display the description and Facebook share button. Each column in the gallery contains an even amount of photos with dynamic sizes and description lengths. Optimizing the flip animation for cross browser testing and creating a dynamic style for past, present and future photos in the gallery were two of the challenges we faced during the creation of this module. Working with the development team on this project gave me quite a bit of experience on API’s, proper documentation, scope management and project planning.

You can view the Canalta Cares Module here Canalta-cares.

Canalta Hotels

Drupal Front- End Styling

The Canalta Hotels website is a project I was tasked with styling based on instructions and designs from the Canalta Hotels team. The site was built using the Drupal 7 so a lot of custom CSS was required in order to recreate the design flats given to our development team. Since we were styling the website based on their designs a lot of attention to detail was required such as accurate measurements, color sampling, converting photos for the web, etc. One challenge we had to work around was how the base Drupal theme the site was built on looked. In order to make it look like the designs we had to manipulate the original theme without breaking the styles in the other parts of the site. I was also tasked with styling the site so it was mobile compatible across different screen sizes.

You can view the Canalta Hotels website here Canaltahotels.com.

Loyalty Engine

Client Administration Console and API

During my time with Loyalty Engine I worked closely with their custom API and administration console for their clients. Loyalty Engine’s API was flexible enough to fit the needs of three different clients but required a lot of tweaking and improvements because of this. One of the new experiences I had in Loyalty Engine was with rapid development. The administration console was used by three different clients so changes would need to be made on three different fronts at their request. The environment was fast and involved a lot of learning and quick development right out of the gate. Other than working with the admin console website I also spent most of my time working with MySQL queries. The Loyalty Engine API uses stored procedures to handle requests from the administration console website and mobile applications. New queries would need to be created whenever a client would request new functionality and existing queries had to be optimized or tweaked when new changes are introduced to the console. While extremely challenging, working with Loyalty Engine quickly provided me with a lot of experience in multiple different fields of web development.

GnuPG

File Import and Decryption

The largest project I was tasked with during my time with Loyalty Engine was the creation of a file import system that downloads multiple .gpg files from a client server using SFTP, decrypt them, import them to the Loyalty Engine database and archive them. The script was written in Shell which used a PHP script as an importer. Originally the script was planned to be written in PHP but due to the sizes of the encrypted files the toll on the server was too large, so an alternative solution was needed. This project was my first experience working with Shell scripting. Since this was an important piece for Loyalty Engine and needed to be completed as soon as possible I quickly familiarized myself with Shell, Linux and GnuPG in the span of a couple of days in order to complete the task given to me. This script led to the removal of a lot of manual process that needed to be done previously before its creation and automation.

Kim Huynh Artworks and Writting

Website Redesign

Kim Huynh is an Associate Professor for the Department of Art at the University of Calgary. Kim uses her website to upload various types of art and writing for personal and professional use. This project started off as a simple updating of her WordPress website, I was asked to update the many plugins the site used and the WordPress CMS itself. After some research and discovery I found that many of the plugins that Kim's website was using were obsolete or not fully compatible with the latest version of WordPress. I discussed these issues with Kim and she decided that she was ready to have the entire website reworked visually and for a more user friendly interface which she can use to update the website herself.

You can view Kim Huynh's website here kimhuynh.ca.
